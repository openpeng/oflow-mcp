# 爆款视频封面设计规则

## 核心原则

1. **3 秒定生死**：封面必须让目标用户在 0.5 秒内理解「这视频讲什么」。
2. **信息密度低**：画面元素不超过 3 个，文字不超过 2 行。
3. **冲突前置**：把最吸引人的冲突、结果或疑问放在主标题。

## 标题公式

| 类型 | 示例 |
|------|------|
| 反常识 | "AI 做视频，不是越像人越好" |
| 数字 | "3 步把文章变视频" |
| 利益 | "看完这个，少加班 2 小时" |
| 情绪 | "我真的服了这届 AI" |
| 悬念 | "为什么你做的视频没人看？" |

## 字数限制

- 主标题：最多 8 个字。
- 副标题：最多 6 个字。
- 总字数越少越好。

## 视觉要素

- 主体人物/物体放在画面中心或黄金分割点。
- 背景简洁，避免与文字撞色。
- 文字描边或阴影保证可读性。
- 配色与视频风格统一。
- 竖屏 9:16 时，标题放在中上部，避开 LibTV 播放时的 UI 遮挡。

## 常见错误

- 标题太长，缩略图看不清。
- 画面杂乱，没有视觉焦点。
- 封面与视频内容无关，导致完播率低。
- 文字颜色与背景相近。
