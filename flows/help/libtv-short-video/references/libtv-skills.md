# LibTV Skill 速查

LibTV 平台已有 100+ 专业 Skill。本工作流在分析环节会根据文章内容推荐一种风格，分镜/封面生成阶段会调用对应视觉风格。

## 常用 Skill 与适用场景

| Skill 名称 | 适用内容 | 视觉特征 |
|-----------|---------|---------|
| 公众号转爆款短视频 | 公众号文章 → 口播短视频 | 真实感、复用原文图片、自动生成脚本分镜 |
| 古典武侠电影全流程导演 | 武侠、江湖、古风剧情 | 电影感、动作分镜、古风调色 |
| 皮克斯动画广告 | 产品广告、品牌 TVC | 3D 卡通、高饱和、故事化 |
| 游戏实机 PV | 游戏宣传、战斗 PV | 写实/幻想场景、节奏感强 |
| 无厘头喜剧 | 搞笑短剧、脑洞创意 | 夸张、节奏快、反转 |
| 新中式美学 TVC | 国风产品、文化内容 | 东方美学、留白、水墨/工笔 |
| 活人感美妆产品测评 | 美妆测评、种草 | 自然光、真实感、细节特写 |
| 精品女频一键成片 | 爽剧、女频剧情 | 剧情化、情绪强、高颜值 |
| 知识口播 | 科普、干货、观点 | 简洁、信息密度高、配图清晰 |

## 选择原则

1. 先尊重用户显式指定的 `style`。
2. 未指定时，根据文章主题、情绪钩子、目标受众匹配。
3. 知识/干货/教程类 → 知识口播。
4. 产品/品牌类 → 皮克斯动画广告 / 新中式美学 TVC。
5. 剧情/故事类 → 古典武侠 / 无厘头喜剧 / 女频爽剧。
6. 游戏/动作类 → 游戏实机 PV。
